This project explores a deeper understanding and use of functions and nested loops.

The "isupper" task checks for uppercase character.

The "isdigit" task checks for a digit (0 through 9).

The "Collaboration is multiplication" task multiplies two integers.

The "The numbers speak for themselves" task prints the numbers, from 0 to 9, followed by a new line.

The "I believe in numbers and signs" task prints the numbers, from 0 to 9, followed by a new line.

The "Numbers constitute the only universal language" task prints 10 times the numbers, from 0 to 14, followed by a new line.

The "The shortest distance between two points is a straight line" task draws a straight line in the terminal.

The "I feel like I am diagonally parked in a parallel universe" task draws a diagonal line on the terminal.

The "You are so much sunshine in every square inch" task prints a square, followed by a new line.

The "Fizz-Buzz" task prints the numbers from 1 to 100, followed by a new line.
But for multiples of three print Fizz instead of the number and for the multiples of five print Buzz.
For numbers which are multiples of both three and five print FizzBuzz.

The "Triangles" task prints a triangle, followed by a new line.
