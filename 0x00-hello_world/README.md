The 0x00.C - Hello, World project contains tasks that practicalise the compilation process within the C language.

The "Peprocessor" task is a script that runs a C file whose name will be saved in a variable called $CFILE through the preprocessor and save the result into another file called c.

The "Compiler" task is a script that compiles a C file but does not link where the output file's name is the same as the C file, but with the extension .o instead of .c

The "Assembler" task is a script that generates the assembly code of a C code and save it in an output file, where the output file's name is the same as the C file, but with the extension .s instead of .c

The "Name" task is a script that compiles a C file and creates an executable named cisfun.

The "Hello, puts" task is a script that prints exactly "Programming is like building a multilingual puzzle, followed by a new line using the function puts, with no use of printf and a return value of 0.

The "Hello, printf" task is a script that prints exactly with proper grammar, but the outcome is a piece of art,, followed by a new line using the function printf, without using puts and a return value of 0.

The "Size is not grandeur, and territory does not make a nation" task is a script that prints the size of various types on the computer it is compiled and run on.
